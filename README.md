Kria SOM Base HW Projects

To generate an XSA for starter kits
1. cd to <part_name>_starter_kits/<starter_kit_name>
2. Run command 
	make xsa  
   Note you can also pass arguments to the make command
	make xsa JOBS=<num> PROJ_NAME=<proj_name>
